# **App Name**: StudyAI

## Core Features:

- User Authentication: Allow users to log in and manage their accounts.
- Subject and Deadline Input: Enable users to specify the subject they are studying and the deadline for their exam.
- Study Schedule Generation: Generate a study schedule based on the subject, deadline, and sections of the material.
- Section Input: Let the user specify what project sections they'd like to include in their study.
- Study Method Selection: Allow users to select preferred study methods (e.g., practice quizzes, reading sections).
- AI-Powered Study Tips: Provide personalized study tips and strategies powered by Gemini based on the entered details. LLM as a tool provides dynamic tips based on the current user inputs.
- Interaction Storage: Store each interaction and generated study plan in Firestore for future access.

## Style Guidelines:

- Primary color: A calm blue (#64B5F6) to promote focus and concentration.
- Background color: Light, desaturated blue (#E3F2FD) for a clean and uncluttered interface.
- Accent color: A muted yellow (#FFD54F) to highlight important elements and calls to action.
- Font pairing: 'Inter' (sans-serif) for body text, 'Space Grotesk' (sans-serif) for headlines.
- Use simple, clean icons to represent study-related actions and concepts.
- A clear and structured layout to provide a good user experience, with well-defined sections for input, schedule display, and AI tips.
- Subtle animations to confirm user interactions and enhance engagement without being distracting.
